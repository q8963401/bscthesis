Random Notes on reading trough “Reasoning about unvertainty”

Allgemeines

  • Wie kann ungewissheit / unvertainty repräsentiert werden?
    • Es gibt verschiedem Ansätze, z.B. DempsterSchäfer belief functions, possibility measures, ranking functions
  • Die meisten Repräsentationen von Ungewissheit starten mit Möglichen Welten (manchmal auch “states” oder “elementary outcomes” genannt)
  • Möglichen Welten sind die Ergebnisse die ein Agent für möglich hält.
    • Die möglichen Welten des Agenten “Werfer einer Münze” besteht aus 6 möglichen Welten, d.h. W: = {w1, w2, w3, w4, w5, w6} , d.h. die Welt wi ist die Welt in welcher der Würfel auf i landet.
    • Die _bekannten Objekte_ werden Events genannt. Ein Event ist eine Menge möglicher Welten.
  • Sei eine Menge W möglicher Welten.
    • Angenommen die Ungewissheit eines Agenten wird durch W’ ⊆ W bezeichnet.
    • Der Agent hält U für möglich wenn U∩W’ ≠ ∅ gilt, d.h. wenn der Agent mindestens ein Event für möglich hält welches in U ist.
    • Der Agent weiß U wenn W’ ⊆ U, d.h. wenn in allen Welten welche der Agent für möglich hält Ugilt. Um es anders auszudrücken: Ein Agent weiß Uwenn er U nicht für möglich hält.
    • Was ein Agent weiß hängt davon
  • Eine formale Logik besteht aus einer Syntax (z.B. einer Sprache) und einer Semantik, d.h. Modellen welche benutzt werden um zu entscheiden ob Formeln in der Sprache wahr oder falsch sind.

Wahrscheinlichkeits Maß

DEF_2.2.1 Eine Algebra über W ist eine Menge ℱvon Teilmengen von W welche W enthält und unter ∩ und ∪ abgeschlossen ist. D.h. sind V, U ∈ ℱ, so auch V∪U ∈ ℱ und V∩U ∈ ℱ. Eine σ-Algebra ist abgeschlossen unter Komplement und aufzählbarer Vereinigung, d.h. sind U1, U2, ⋯, Un ∈ ℱ so auch ∪i ∈ {1, 2, ⋯, n}Ui.
Minimal Ein U ∈ ℱ ist genau dann minimal, wenn U ≠ ∅ und U’ ⊆ U ⇒ U’ = ∅
DEF_2.2.2 Ein probalistischer Raum ist ein ein tuple (W, ℱ, μ), wobei ℱeine algebra über W ist und μ:F → [0, 1]die folgenden Bedingungen erfüllt
  • P1 μ(W) = 1
    P2 μ(U∪V) = μ(U) + μ(V) wenn U und V disjunkte Elemente von ℱ sind.
    Bemerkung Die Mengen in ℱwerden messbare Mengen genannt, μ wird Wahrscheinlichkeitsmaß auf W (oder über ℱmit F ≠ 2W) genannt. Die Argumente von μ sind keine Elemente von W sondern Teilmengen aus W.

Multi-Agenten Systeme

Epistemic-Frame Ein epistemic Frame F ist ein Paar (W, K) wobei W die Mengen der möglichen Welten darstellt und K ⊆ W × W eine binäre Relation auf W ist. K wird manchmal als Möglichkeitsrelation oder Erreichbarkeitsrelation bezeichnet wird. Intuitiv gesprochen ist (w, w’) ∈ K genau dann, wenn der Agent w’ für eine mögliche Welt in w hält.
Möglichkeitsrelation K(w) = {w’:(w, w’) ∈ K}. Es ist K(w) die Menge der Welten die ein Agent in der Welt w für möglich. Sei bspw. W = {{w1, w2, w3, w4, w5, w6}, {w1}, {w6}} die Menge der Welten wobei w1...w6 für die Augenfarben eines Würfels stehen und K = {({w1, w2, w3, w4, w5, w6}, {w1}), ({w1, w2, w3, w4, w5, w6}, {w6})} also K({w1, w2, w3, w4, w5, w6}) = {{w1}, {w6}}. Dann hält der Agent es einzig für möglich, dass eine 1 oder eine 6 gewürfelt wird ;-)
Die Frage ob ein Agent ein Event für möglich hält oder ein Event weiß, hängt von der Welt ab.
  • Ein Agent hält U für möglich in der Welt w (in einem epistemic-frame), genau dann wenn U∩K(w) ≠ ∅.
  • Der Agent weiß U in der Welt w, wenn K(w) ⊆ U. D.h. wenn jede Welt die er für möglich hält in U liegt.
Es lassen sich einige Standardsätze der Relationentheorie über K formulieren (eine Auswahl):
Reflexiv K wird reflexiv genannt, wenn für alle w ∈ W gilt (w, w) ∈ K.
Euklidisch K wird euklidisch genannt, wenn aus (u, v), (u, w) ∈ K folgt, dass (v, w) ∈ K für alle u, v ∈ K.
Seriell K wird seriell genannt wenn es für jedes w ∈ W ein w’ ∈ W gibt, so dass (w, w’) ∈ K ist. Dies lässt sich so interpretieren, dass der Agent immer irgendeine Welt für möglich hält.
Transitiv K wird transitiv genannt wenn aus (u, v), (v, w) ∈ K folgt, dass (u, w) ∈ K für alle u, v, w ∈ K.
Äquivalent K ist eine Äquivalenzrelation, wenn K reflexiv, symmetrisch und transitiv ist.
Wie werden mehrere Agenten repräsentiert? Ein epistemic-frame F für n Agenten ist ein Tupel (W, K1, ⋯,  Kn) wobei jedes Ki eine binäre Relation auf W ist. Ki(w) ist entsprechend wieder die Menge der Welten die Agent i in w für möglich hält.
Ein Beispiel:
  • Sei ein Kartenspiel mit 3 Karten A, B und C und die Welt W = {A, B, C} sowie 2 Agenten, Agent 1 mit K1 und Agent 2 mit K2.
  • Jeder Agent hat eine Karte, weiß aber nicht welche Karte der andere Agent hat, die dritte Karte liegt verdeckt auf dem Tisch.
  • Eine mögliche Welt wird dadurch charakterisiert, welcher Agent welche Karte hält, d.h. hat Agent 1 Karte A und Agent 2 Karte B so lautet die mögliche Welt (A, B).
  • Es sind 6 mögliche Welten denkbar: (A, B), (A, C), (B, A), (B, C), (C, A), (C, B).
  • Hat bspw. Agent 1 die Karte A und weiß nicht welche Karte Agent 2 hat, so hält Agent 1 die Welten (A, B) und (A, C) für möglich, d.h. es ist K1({A}) = {B, C}. Umgekehrt hält Agent 2 die Welten (B, A), (B, C) für möglich bzw. K2({B}) = {A, C}

Logics for reasoning about Uncertainty

Propositional logic:
  • Set of primitive propositions Φ ≠ ∅ is a vocabulary that contains fixed statements that predicate things like „it is raining“ or „today is monday“
  • ℒProp(Φ) is the set of all formulas that can be built by conjunction and negation of the primitive propositions that are given by the vocabulary Φ.
    • Let φ, ψ ∈ Φ primitive propositions, so are ¬φ, (φ∧ψ), (φ∨ψ), (φ → ψ) ∈ ℒProp(Φ) formulas.
    • There are some abbreviations:
      • φ∨ψ ≡ ¬(¬φ∧¬ψ)
      • φ ⇒ ψ ≡ ¬φ∨ψ
      • φ ⇔ ψ ≡ (φ ⇒ ψ)∧(ψ ⇒ φ)
      • true ≡ φ∨¬φ where φ ∈ Φ is a primitive proposition (tautology)
      • false ≡ ¬true
  • Semantics assigns a truth-value to a given formula. This is acomplished by the truth assignment. The truth assigment is a function that maps the primitive propositions of Φ to a truth value. A truth value is an element of the Set {true, false}.
    • The truth assigment guarantees that each primitive proposition has exactly one truth value. Is either true or false, not true and false and not neither, true nor false. This leads to the two value logic.
      • Truth Assigment: Is a mapping v:Φ → {true, false} where Φ is the set of primitive propositions.
      • Satisfiability under truth assignment: If an arbitrary formula φ is satisfied by a truth assignment v, it is written v⊨φ
    • v⊨φ determines the satifiability of the formula φ under the truth asigment v.
  • Etc.

3 Türen, eine Ziege Problem

Es stehen 3 verschlossene Türen zur Auswahl. Hinter 2 der 3 Türen verbergen sich Schafe, hinter einer Tür eine Auto. Man darf sich eine Tür auswähle und wählt Tür 1. Der Quizmaster (welcher weiß was hinter den Türen ist) öffnet Tür 3 und es kommt ein Schaf zum Vorschein. Nun darf man, wenn man möchte sich eine neue Tür auswählen. Was sollte man machen? Bei der Tür 1 bleiben oder zur Tür 2 wechseln? Was ist nun die Wahrscheinlichkeit, dass sich das Auto hinter Tür 1 verbirt? 1/3 oder 2/3?
